aethernum.io — CDK Infrastructure

S3 + CloudFront + Route53 + ACM — single stack, us-east-1

Key Design Decisions

  • • www redirect via CloudFront Function (no extra S3 bucket)
  • • Single distribution, single certificate, single stack
  • • us-east-1 — avoids ACM cross-region complexity
  • • Price Class 100 — ~40% egress savings (US/CA/EU)
  • • S3 versioning + RETAIN — rollback + data safety

Cost Estimate

  • • CloudFront: Free tier (1 TB egress, 10M requests)
  • • S3: ~$0.023/GB storage + ~$0.005/1K PUT requests
  • • Route53: $0.50/month hosted zone + $0.40/M queries
  • • ACM: Free (public certificates)
  • • CloudFront Functions: Free tier (2M invocations)
  • • Total: ~$0.55–0.75/month

Security Posture

  • • S3: private — Block Public Access ALL, no bucket policy
  • • CloudFront OAC — only auth path to S3 origin
  • • HTTPS-only via ACM — TLS 1.2 minimum
  • • HSTS preload-ready: max-age=63072000; includeSubDomains
  • • Deploy via OIDC — no long-lived IAM credentials
  • • Least-privilege IAM: s3:PutObject + cloudfront:CreateInvalidation
